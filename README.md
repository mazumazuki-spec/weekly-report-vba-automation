# Weekly Report Automation with VBA

## Project Overview
This project demonstrates a VBA-assisted workflow for preparing weekly Excel reports. The process helps consolidate multiple downloaded reports, clean recurring data issues, and generate a standardized summary output.

## Problem
Weekly reports required multiple manual steps, including downloading reports, combining files, cleaning data, checking values, and preparing the final output.

## Solution
Developed Excel VBA macros to automate repetitive steps such as:
- Combining multiple Excel files into one workbook
- Cleaning and standardizing report structure
- Removing non-transaction rows
- Adding date and outlet information
- Creating a final summary sheet for weekly reporting

## Workflow
Manual Download Reports → Run VBA Macro → Combine Files → Clean Data → Create Summary Sheet → Weekly Report Output

## Impact
Reduced weekly report preparation time to approximately 1.5 days and improved workflow consistency.

## Tools
Excel VBA, Macro Automation, Excel Formula, Data Cleaning
